# Cremina Lever Machine

| Mező | Érték |
|---|---|
| Profil jellege | Klasszikus, karos kávégép karaktert utánzó profil |
| Kávé | klasszikus, testes espresso |
| Pörkölés | közepes-sötét / sötét |
| Várható karakter | édes, kerek, alacsony sav, csokis-diós |
| Arányok | 1:1.5–1:2.2 |

**Setup:**
- **Gaggia Classic Pro + Gaggiuino**
- **DF64V Gen 2**
- **SSP Sweet Lab Espresso V3**
- **IMS B682TH24.5M**

## Alap recept

| Paraméter | Érték |
|---|---:|
| Dózis | 18g |
| Cél hozam | 27-40g |
| Teljes profilidő | 40-60s |
| Hőmérséklet | 86.5°C |
| Őrlés | Nagyon finom |

## Fázisok

- **Pre-infusion & Soak:** Alacsony nyomású előáztatás és hosszabb áztatás.
- **Ramp-up:** Fokozatos nyomásfelépítés 9 bar köré.
- **Ramp-down:** Hosszú, csökkenő nyomásfázis 3 bar felé.

## Dial-in logika

- **Mikor válaszd ezt?** Ha közepes-sötét vagy sötét pörkölésű, brazil, csokis-diós, klasszikus olaszosabb, alacsonyabb savú, testesebb kávéd van.
- **Ízben várható:** Több test, alacsonyabb savérzet, csokisabb, diósabb, karamellesebb karakter.
- **Mikor ne ezt válaszd?** Világosabb, gyümölcsös specialty kávékhoz (pl. Kenya, Burundi, gyümölcsös Colombia), mert ezeknél tompa, nehéz lehet a végeredmény.
