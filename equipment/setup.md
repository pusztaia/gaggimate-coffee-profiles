# Equipment Setup

**Gép:** Gaggia Classic Pro 2025 + GaggiMate Pro  
**Daráló:** DF64V Gen 2  
**Kések:** SSP Sweet Lab Espresso V3  
**Kosár:** IMS B682TH24.5M  
**RPM:** 1200 baseline  
**RPM tartomány:** 800-1800 RPM, változtatható  
**Őrlőskála:** 0-90, egész jelölések  
**Mérleg státusz:** jelenleg nincs BOOKOO Themis Ultra; a profilok idő/fázis alapúak, nem aktív brew-by-weight profilok.

## Fontos szabályok

- A DF64V Gen 2 skálán tizedes értéket nem kezelünk stabil beállításként.
- Profilonként a JSON a gépi forrás, az MD az olvasható recept.
- Az MD fájlokban nincs beágyazott JSON blokk.
- Shot logot minden érdemi tesztről érdemes felvenni.
- Puck screen használatakor a screen legyen tiszta és száraz.
- A célhozamot jelenleg külön mérlegen figyeld; automatikus BOOKOO / Bluetooth mérleges stop később opcionális.
