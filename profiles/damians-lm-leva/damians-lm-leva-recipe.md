# Damian's LM Leva

**Profil jellege:** La Marzocco Leva jellegű, modern lever / pressure profiling irány.

**Kávé:** modern specialty espresso
**Pörkölés:** világos-közepes / közepes
**Várható karakter:** tiszta, gyümölcsösebb, hosszabb lefolyás
**Arányok:** 1:2.0–1:2.5

**Setup:**
- **Gaggia Classic Pro + Gaggiuino**
- **DF64V Gen 2**
- **SSP Sweet Lab Espresso V3**
- **IMS B682TH24.5M**

## Alap recept

| Paraméter | Érték |
|---|---:|
| Dózis | 18g |
| Cél hozam | 36-45g |
| Teljes profilidő | 20-45s |
| Hőmérséklet | 89°C |
| Őrlés | Nagyon finom |

## Fázisok

- Hosszabb fill / preinfusion fázis.
- Fokozatos nyomásfelépítés.
- Rövid, magasabb nyomású szakasz.
- Hosszú, csökkenő nyomású (declining pressure) szakasz.

## Dial-in logika

- **Mikor válaszd ezt?** Világos-közepes és közepes specialty kávékhoz, mosott afrikai, gyümölcsös latin-amerikai, teás, virágos, savasabb kávékhoz. Ez a profil áll a legközelebb a gyümölcsös specialty profilokhoz (pl. Wangera, Kirinyaga, Burundi).
- **Ízben várható:** Tisztább gyümölcsösség, kevésbé agresszív sav, jobb édesség, modernebb specialty espresso karakter.
- **A te kávéidhoz:** Ez a profil passzol a legjobban a gyümölcsös specialty kávéidhoz.
